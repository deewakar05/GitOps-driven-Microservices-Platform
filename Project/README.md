# GitOps-Driven Microservices Platform with Full Observability

A comprehensive DevOps platform implementing GitOps principles with full observability stack.

## Architecture Overview

- **CI/CD**: GitHub Actions
- **Container Registry**: AWS ECR
- **Infrastructure**: Terraform
- **Orchestration**: AWS EKS
- **GitOps**: ArgoCD
- **Monitoring**: Prometheus + Grafana
- **Logging**: Elastic Cloud on Kubernetes (ECK)

## Project Phases

- **Phase 1**: Foundation Setup (Current)
  - Project structure
  - Sample microservice
  - Docker configuration
  - GitHub Actions CI
  - Terraform for AWS ECR

- **Phase 2**: Infrastructure Provisioning
  - EKS cluster setup
  - Networking configuration
  - IAM roles and policies

- **Phase 3**: GitOps with ArgoCD
  - ArgoCD installation
  - Application definitions
  - Sync policies

- **Phase 4**: Observability Stack
  - Prometheus setup
  - Grafana dashboards
  - ECK for logging

- **Phase 5**: Advanced Features
  - Service mesh (optional)
  - Security hardening
  - Multi-environment setup
